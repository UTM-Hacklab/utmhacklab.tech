    # 项目

    UTM的Hacklab为做项目的学生们提供了场所和指导。我们通过建立探索或实施项目支持学生们去探索他们热爱的技术领域。

    # UTM谷歌社区项目

    UTM的谷歌开发者学生社团今年启动了一个社区项目计划。

    对软件开发感兴趣的学生们与高年级导师和大学教授一起为展示谷歌社区项目建立项目

    # 学生项目

    Hacklab为从事课程之外项目的学生提供了技术支持和指导。

    如果你也想参与到这些项目，你可以在Outlook或Microsoft Teams上通过邮件与这些学生项目贡献者们联系。

    如果你想建立一个自己独特的项目， 你可以：

    • 与我们的组长在Microsoft Teams上的数字社区上联系

    • 拜访我们位于Deerfield Hall，2014号教室的UTM办公室 查看我们的办公时间
